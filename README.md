# Упрощённый аналог утилиты telnet

Данная программа реализует упрощённый аналог утилиты `telnet` для подключения к TCP-серверу, реализованная на языке Go.  
Она принимает хост, порт и опционально таймаут.

---

## Использование

### Параметры:

- `host` - хост или доменное имя
- `port` - порт
- `timeout` - время подключения, по умолчанию 10 секунд

---

### Примечания:

- При нажатии Ctrl+D клиент закрывает соединение

---

### Серверы для теста:

- smtp.freesmtpservers.com 25
- tcpbin.com 4242

---

## Makefile

Реализованы следующие цели для `make`:

- `make lint`  
  Запускает go vet и golangci-lint по всему проекту.
- `make test`  
  Запускает интеграционный тест.
